 # 💻 C# MyFinancialCrm_Project 💻

MyFinancialCrm projesi Murat Yücedağ'ın "C# Eğitim Kampı" eğitiminde, caselerle geliştirilmiş bir Finansal CRM uygulamasıdır. 

## 👩‍💻 Kullanılan Teknolojiler 
##### ⚡ MSSQL Server
##### ⚡ C# (.NET Framework)
##### ⚡ Entity Framework
##### ⚡ Windows Forms 

## 👩‍💻 Projedeki İçerikler
##### ✅ Kullanıcı Girişi
##### ✅ Banka İşlemleri
##### ✅ Windows Forms Arayüzü
##### ✅ MSSQL ile Veritabanı Entegrasyonu
